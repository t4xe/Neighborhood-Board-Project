Neighborhood Community Board

Backend API is running. Frontend (Angular) is to be implemented by the frontend developer.

Use /api as the base URL for all API calls. See API documentation (or docs/API.md in the repo).

  • POST /api/authentication/login — sign in
  • GET /api/categories — list categories
  • GET /api/posts — list posts (pagination, filter by category, zone)
  • GET /api/users/me — current user (requires auth)
